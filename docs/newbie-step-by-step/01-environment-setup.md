# 01 - Environment Setup

Target: setup enviroment to build firmware.

## Tool needed

- arm-none-eabi-gcc
- make
- st-flash
- VS Code + C/C++ extension

## Fast test

```bash
arm-none-eabi-gcc --version
make --version
st-flash --version
```

If miss any enviroment, please Install it first

## Open Project

```bash
git clone <repo-url>
cd DUNGEON-GAME
```
